# zc_og05 (tir game)
## игра тир
### Принцип игры  
1. На игровом поле в случайной позиции появляется цель 
      ![](img/target.png) 
2. Игрок кликает мышью по цели
3. При попадании цель перемещается в другую случайную позицию
